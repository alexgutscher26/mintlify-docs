<Info>
  **Prerequisites**: 
  - Node.js (version 19 or higher)
  - Git installed on your system
  - A code editor (VS Code recommended)
  - Basic knowledge of terminal/command line
</Info>

Follow these steps to set up the MonitorFlow documentation environment:

**Step 1**: Clone the MonitorFlow repository and install Mintlify:

<CodeGroup>

```bash npm
# Clone the repository
git clone https://github.com/your-org/monitorflow.git
cd monitorflow

# Install Mintlify CLI
npm i -g mintlify
```

```bash yarn
# Clone the repository
git clone https://github.com/your-org/monitorflow.git
cd monitorflow

# Install Mintlify CLI
yarn global add mintlify
```

</CodeGroup>

**Step 2**: Navigate to the documentation directory and start the development server:

```bash
mintlify dev
```

A local preview of your documentation will be available at `http://localhost:3000`.

### Custom Ports

By default, Mintlify uses port 3000. You can customize the port Mintlify runs on by using the `--port` flag. To run Mintlify on port 3333, for instance, use this command:

```bash
mintlify dev --port 3333
```

If you attempt to run Mintlify on a port that's already in use, it will use the next available port:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Mintlify Versions

Please note that each CLI release is associated with a specific version of Mintlify. If your local website doesn't align with the production version, please update the CLI:

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## Validating Links

The CLI can assist with validating reference links made in your documentation. To identify any broken links, use the following command:

```bash
mintlify broken-links
```

## Troubleshooting

<AccordionGroup>
  <Accordion title='Error: Could not load the "sharp" module'>
    This may be due to an outdated version of node. Try the following:
    1. Remove the current mintlify installation: `npm remove -g mintlify`
    2. Ensure Node.js version is 19 or higher: `node --version`
    3. Reinstall mintlify: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Environment Setup Issues">
    1. Verify you're in the correct directory with `docs.json`
    2. Check Node.js version compatibility
    3. Clear the mintlify cache: delete the ~/.mintlify folder
    4. Run `mintlify dev` again
  </Accordion>

  <Accordion title="MonitorFlow Specific Issues">
    1. Ensure all required environment variables are set
    2. Verify the docs.json configuration is correct
    3. Check that all referenced images exist in the /images directory
    4. For API documentation issues, verify the OpenAPI spec is valid
  </Accordion>
</AccordionGroup>

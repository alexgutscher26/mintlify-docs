## Overview

This quickstart guide will help you set up ReplyIQ and generate your first AI-powered content in under 5 minutes.

<Info>
  **Prerequisites**: You'll need a modern web browser. <b>API keys for AI services (OpenAI, Mistral, Google, etc.) are required to generate content.</b> See below for details on obtaining these keys.
</Info>

<Warning>
  <b>Security Note:</b> Never share your API keys publicly. Treat them as sensitive credentials.
</Warning>

## Step 1: Create Your Account

<Steps>
  <Step title="Sign Up">
    Visit [replyiq.com](https://replyiq.com) and create your account using:
    - Email and password
    - Google OAuth
    - GitHub OAuth
    - Discord OAuth
  </Step>
  <Step title="Verify Email">
    Check your email and click the verification link to activate your account
  </Step>
  <Step title="Complete Profile">
    Fill in your profile information and select your use case (individual, team, business)
  </Step>
</Steps>

## Step 2: Configure AI Services

<Warning>
  AI service configuration is required to generate content.<br/>
  <b>Important:</b> The first user to sign up automatically becomes an admin. Make sure this is a trusted account.
</Warning>

<Steps>
  <Step title="Access Admin Dashboard">
    Navigate to **Dashboard â†’ Settings â†’ General â†’ AI Model Provider**
  </Step>
  <Step title="Add API Keys">
    Configure your preferred AI services. <b>You'll need to obtain API keys from each provider's website:</b>
    - <b>OpenAI:</b> [https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)
    - <b>Mistral AI:</b> [https://console.mistral.ai/api-keys/](https://console.mistral.ai/api-keys/)
    - <b>Google AI (Gemini):</b> [https://makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)
    
    <Tabs>
      <Tab title="OpenAI">
        ```json
        {
          "provider": "openai",
          "apiKey": "sk-...",
          "models": ["gpt-4", "gpt-3.5-turbo"]
        }
        ```
      </Tab>
      <Tab title="Mistral">
        ```json
        {
          "provider": "mistral",
          "apiKey": "...",
          "models": ["mistral-large", "mistral-medium"]
        }
        ```
      </Tab>
      <Tab title="Google AI">
        ```json
        {
          "provider": "google",
          "apiKey": "AIza...",
          "models": ["gemini-pro", "gemini-pro-vision"]
        }
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Test Connection">
    Click **Test Connection** to verify your API keys are working correctly
  </Step>
  <Step title="Save Configuration">
    Save your settings and set default models for different use cases
  </Step>
</Steps>

## Step 3: Install Browser Extension

<CardGroup cols={3}>
  <Card title="Chrome" icon="chrome">
    1. Visit [Chrome Web Store](https://chrome.google.com/webstore)
    2. Search for "ReplyIQ"
    3. Click "Add to Chrome"
    4. Pin the extension to your toolbar
    5. <b>Note:</b> The extension may not be available in all regions. Check for updates after installation.
  </Card>
  <Card title="Firefox" icon="firefox">
    1. Visit [Firefox Add-ons](https://addons.mozilla.org)
    2. Search for "ReplyIQ"
    3. Click "Add to Firefox"
    4. Grant necessary permissions
    5. <b>Note:</b> The extension may not be available in all regions. Check for updates after installation.
  </Card>
  <Card title="Edge" icon="edge">
    1. Visit [Edge Add-ons](https://microsoftedge.microsoft.com/addons)
    2. Search for "ReplyIQ"
    3. Click "Get"
    4. Enable the extension
    5. <b>Note:</b> The extension may not be available in all regions. Check for updates after installation.
  </Card>
</CardGroup>

### Extension Setup

<Steps>
  <Step title="Sign In">
    Click the ReplyIQ extension icon and sign in with your account
  </Step>
  <Step title="Grant Permissions">
    Allow the extension to access social media sites when prompted
  </Step>
  <Step title="Verify Connection">
    The extension should show "Connected" status with your account info
  </Step>
</Steps>

## Step 4: Generate Your First Content

<Steps>
  <Step title="Access the Dashboard">
    Go to your ReplyIQ dashboard and select one of the AI tools from the sidebar
  </Step>
  <Step title="Choose a Tool">
    Select the tool that matches your content needs:
    - **Hashtag Generator**: Create relevant hashtags for your posts
    - **Image Caption Generator**: Generate compelling image captions
    - **Language Translator**: Translate content to other languages
    - **Sentiment Analysis**: Analyze the emotional tone of text
    - **Story Generator**: Create engaging social media stories
    - **Thread Generator**: Generate coherent thread content
    - **Video Script Generator**: Create scripts for video content
  </Step>
  <Step title="Configure Your Content">
    <Tabs>
      <Tab title="Tone Selection">
        Choose from available tones:
        - **Professional**: Formal, business-appropriate
        - **Casual**: Relaxed, conversational
        - **Friendly**: Warm, approachable
        - **Humorous**: Light-hearted, witty
        - **Supportive**: Encouraging, empathetic
      </Tab>
      <Tab title="Content Parameters">
        Set specific parameters based on the selected tool:
        - **Length**: Short, medium, or long content
        - **Format**: Structure and style options
        - **Target Platform**: Optimize for specific social platforms
        - **Language**: Select output language
      </Tab>
    </Tabs>
  </Step>
  <Step title="Generate Content">
    Enter your input text or parameters and click the **Generate** button
  </Step>
  <Step title="Review and Edit">
    - Review the generated content
    - Make any necessary edits
    - Save or copy the content for use
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Content generation failing">
    1. Check your AI service configuration in the dashboard
    2. Verify your API keys are valid and have sufficient credits
    3. Try switching to a different AI model
    4. Contact support if the issue persists
  </Accordion>

  <Accordion title="Slow response times">
    1. Check your internet connection
    2. Try using a different AI model (some are faster than others)
    3. Reduce the complexity or length of your requested content
    4. Clear your browser cache and restart your browser
  </Accordion>

  <Accordion title="Browser extension issues">
    1. Verify the extension is properly installed and enabled
    2. Check if you're signed in to your ReplyIQ account
    3. Try disabling and re-enabling the extension
    4. Clear your browser cache and cookies
    5. <b>Still stuck?</b> Visit our [FAQ](/support/faq) or [contact support](mailto:support@replyiq.com)
  </Accordion>
</AccordionGroup>

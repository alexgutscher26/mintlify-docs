## Setup your environment

Learn how to set up MonitorFlow and start monitoring your SaaS events.

### Prerequisites

<AccordionGroup>
  <Accordion icon="tools" title="Install required tools">
    Before you begin, ensure you have the following installed:
    - Node.js 18+
    - npm or pnpm
    - PostgreSQL database i use neondb
    - Discord account
    - Stripe account (for payments)
  </Accordion>
  <Accordion icon="git" title="Clone the repository">
    Get started by cloning the MonitorFlow repository:
    ```bash
    git clone https://github.com/alexgutscher26/MonitorFlow.git
    cd MonitorFlow
    ```
  </Accordion>
</AccordionGroup>

### Initial Configuration

<AccordionGroup>
<Accordion icon="gear" title="Environment setup">
  1. Install dependencies:
  ```bash
  pnpm install
  ```
  2. Set up your environment variables:
  ```bash
  cp .env.example .env
  ```
  3. Configure your `.env` file with:
  - Database URL
  - Clerk API keys
  - Discord bot token
  - Stripe API keys
</Accordion>
<Accordion icon="database" title="Database setup">
  Run database migrations to set up your PostgreSQL database:
  ```bash
  pnpm prisma migrate dev
  ```
</Accordion>
</AccordionGroup>

### Launch MonitorFlow

<AccordionGroup>
<Accordion icon="rocket" title="Start the application">
  Launch the development server:
  ```bash
  pnpm dev
  ```
  Your MonitorFlow instance will be available at `http://localhost:3000`
</Accordion>
<Accordion icon="discord" title="Configure Discord integration">
  1. Create a Discord server for your notifications
  2. Add the MonitorFlow bot to your server
  3. Get api settings in your dashboard
  4. Test the integration with a sample event
</Accordion>
</AccordionGroup>

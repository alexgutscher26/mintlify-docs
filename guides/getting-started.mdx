## Step 2: AI Model Configuration

<Warning>
  **Important**: AI model configuration is required before you can generate replies. This is done through the admin dashboard, not environment variables.
</Warning>

### Setting Up AI Providers

<Steps>
  <Step title="Access Settings">
    Navigate to **Dashboard â†’ Settings â†’ AI Model Provider**
  </Step>
  <Step title="Choose Your AI Provider">
    <Tabs>
      <Tab title="OpenAI (Recommended)">
        **Best for:** General use, high quality responses
        
        1. Get API key from [OpenAI Platform](https://platform.openai.com)
        2. Add key to ReplyIQ dashboard
        3. Select default models (GPT-4, GPT-3.5 Turbo)
        4. Test connection
        
        **Models available:**
        - GPT-4 Turbo (best quality)
        - GPT-4 (high quality)
        - GPT-3.5 Turbo (fast & cost-effective)
      </Tab>
      <Tab title="Anthropic Claude">
        **Best for:** Nuanced, thoughtful responses
        
        1. Get API key from [Anthropic Console](https://console.anthropic.com)
        2. Add key to ReplyIQ dashboard
        3. Select Claude models
        4. Test connection
        
        **Models available:**
        - Claude 3 Opus (highest quality)
        - Claude 3 Sonnet (balanced)
        - Claude 3 Haiku (fastest)
      </Tab>
      <Tab title="Google AI">
        **Best for:** Multimodal understanding
        
        1. Get API key from [Google AI Studio](https://aistudio.google.com)
        2. Add key to ReplyIQ dashboard
        3. Configure Gemini models
        4. Test connection
        
        **Models available:**
        - Gemini Pro (general purpose)
        - Gemini Pro Vision (image understanding)
      </Tab>
    </Tabs>
  </Step>
  <Step title="Test Your Configuration">
    Use the built-in connection tester to verify your API keys work correctly
  </Step>
  <Step title="Set Default Models">
    Choose default models for different scenarios:
    - Professional communications
    - Casual interactions
    - Platform-specific defaults
  </Step>
</Steps>

### Model Selection Guide

<AccordionGroup>
  <Accordion title="For Professional Use (LinkedIn, Business Twitter)">
    **Recommended Models:**
    - GPT-4 or Claude 3 Opus for important communications
    - GPT-3.5 Turbo or Claude 3 Sonnet for regular interactions
    
    **Why:** These models provide nuanced, professional language and understand business context better.
  </Accordion>

  <Accordion title="For Casual Use (Personal Twitter, Facebook)">
    **Recommended Models:**
    - GPT-3.5 Turbo or Claude 3 Haiku for speed
    - GPT-4 for complex or sensitive topics
    
    **Why:** Faster models are sufficient for casual interactions and help manage costs.
  </Accordion>

  <Accordion title="For High-Volume Use">
    **Recommended Models:**
    - Claude 3 Haiku (fastest, most cost-effective)
    - GPT-3.5 Turbo (good balance of speed and quality)
    
    **Why:** These models provide good quality at high speed and lower cost.
  </Accordion>
</AccordionGroup>

## Step 3: Browser Extension Setup

## Step 4: Your First Reply

Let's generate your first AI-powered reply:

<Steps>
  <Step title="Find a Post">
    Navigate to Twitter, LinkedIn, or Facebook and find a post you want to reply to
  </Step>
  <Step title="Start Reply">
    Click the reply button on the post - you should see the ReplyIQ interface appear
  </Step>
  <Step title="Select Tone">
    Choose the appropriate tone for your reply:
    - **Professional** for business content
    - **Casual** for everyday conversations
    - **Friendly** for warm interactions
    - **Supportive** for encouraging responses
  </Step>
  <Step title="Generate Reply">
    Click **"Generate Reply"** and wait for the AI to create a response
  </Step>
  <Step title="Review and Edit">
    - Read the generated reply
    - Make any necessary edits
    - Add emojis if suggested
    - Ensure it matches your voice
  </Step>
  <Step title="Use the Reply">
    Click **"Use Reply"** to insert it into the platform's text field, then post normally
  </Step>
</Steps>

## Common Mistakes to Avoid

<Warning>
  **Don't:** Post AI-generated content without reviewing it first. Always read and edit replies before posting.
</Warning>

<AccordionGroup>
  <Accordion title="Over-relying on AI">
    **Problem:** Using AI for every reply without adding personal touch
    
    **Solution:** 
    - Edit replies to match your voice
    - Add personal experiences or insights
    - Use AI as a starting point, not the final word
  </Accordion>

  <Accordion title="Ignoring Context">
    **Problem:** Not considering the full conversation context
    
    **Solution:**
    - Read the entire thread before replying
    - Consider the relationship with the original poster
    - Adapt tone based on the conversation's mood
  </Accordion>

  <Accordion title="Wrong Tone Selection">
    **Problem:** Using inappropriate tone for the platform or context
    
    **Solution:**
    - Learn platform cultures and expectations
    - Match tone to the original post's tone
    - When in doubt, err on the side of professionalism
  </Accordion>

  <Accordion title="Neglecting Platform Differences">
    **Problem:** Using the same approach across all platforms
    
    **Solution:**
    - Customize approach for each platform
    - Understand platform-specific best practices
    - Adjust content length and style accordingly
  </Accordion>
</AccordionGroup>

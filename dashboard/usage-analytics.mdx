# Usage & Analytics

The Usage & Analytics dashboard provides comprehensive insights into your ReplyIQ usage patterns, content performance, and account limits. Use these metrics to optimize your content strategy and manage your subscription effectively.

## Overview

<Frame>
  ![Usage & Analytics Dashboard](/images/dashboard/usage-analytics.png)
  <Caption>The Usage & Analytics dashboard provides a comprehensive view of your ReplyIQ activity</Caption>
</Frame>

The dashboard is divided into several key sections:

<CardGroup cols={2}>
  <Card title="Usage Summary" icon="chart-pie">
    High-level overview of your account usage across all tools
  </Card>
  <Card title="Content Performance" icon="chart-line">
    Metrics on how your generated content is performing
  </Card>
  <Card title="Usage Trends" icon="chart-bar">
    Historical usage patterns to identify trends
  </Card>
  <Card title="Quota Management" icon="gauge">
    Monitor your subscription limits and usage
  </Card>
</CardGroup>

## Usage Summary

The Usage Summary section provides a quick overview of your ReplyIQ activity:

<AccordionGroup>
  <Accordion title="Total Generations">
    The total number of content pieces generated across all tools in the current billing period.
    
    <Tip>
      Compare this with previous periods to track your usage growth over time.
    </Tip>
  </Accordion>
  
  <Accordion title="Tool Breakdown">
    A breakdown of your usage by specific tools:
    
    - **Hashtag Generator**: Number of hashtag sets generated
    - **Image Caption Generator**: Number of captions created
    - **Language Translator**: Number of translations performed
    - **Sentiment Analysis**: Number of texts analyzed
    - **Story Generator**: Number of stories created
    - **Thread Generator**: Number of threads generated
    - **Video Script Generator**: Number of scripts written
  </Accordion>
  
  <Accordion title="Usage by Time">
    A timeline showing your usage patterns by day, week, or month.
    
    <Warning>
      Sudden spikes in usage may indicate unauthorized access or potential API key exposure.
    </Warning>
  </Accordion>
</AccordionGroup>

## Content Performance

Track how your AI-generated content is performing:

<Steps>
  <Step title="Performance Metrics">
    View key performance indicators for your content:
    
    - **Engagement Rate**: How users interact with your content
    - **Click-Through Rate**: Percentage of users who click on links
    - **Conversion Rate**: Actions taken after engaging with content
    - **Sentiment Score**: Emotional response to your content
  </Step>
  
  <Step title="Content Comparison">
    Compare performance across different:
    
    - Content types
    - Platforms
    - Time periods
    - AI models used
  </Step>
  
  <Step title="Top Performing Content">
    Identify your most successful content pieces to understand what works best.
  </Step>
</Steps>

## Usage Trends

Analyze your usage patterns over time:

<Frame>
  ![Usage Trends Chart](/images/dashboard/usage-trends.png)
  <Caption>Track your usage patterns over time to identify trends and optimize your subscription</Caption>
</Frame>

<Tip>
  Use the date range selector to view trends for different time periods: last 7 days, last 30 days, last 3 months, or custom range.
</Tip>

Key trend metrics include:

- **Daily/Weekly/Monthly Usage**: Track usage patterns over time
- **Peak Usage Times**: Identify when you use ReplyIQ most frequently
- **Growth Rate**: How your usage is changing over time
- **Seasonal Patterns**: Identify cyclical usage patterns

## Export Data

You can export your usage and analytics data for further analysis:

<Steps>
  <Step title="Select Date Range">
    Choose the time period for which you want to export data
  </Step>
  
  <Step title="Choose Export Format">
    Select your preferred format:
    - CSV
    - Excel
    - PDF Report
  </Step>
  
  <Step title="Download">
    Click the "Export" button to download your data
  </Step>
</Steps>

## Custom Reports

Create custom reports to focus on specific metrics that matter to your business:

1. Click the "Create Custom Report" button
2. Select the metrics you want to include
3. Choose visualization types (charts, tables, etc.)
4. Set up automated delivery (optional)
5. Save your report configuration

## API Usage Tracking

For developers using the ReplyIQ API, the dashboard provides detailed API usage metrics:

- **API Calls**: Total number of API requests
- **Endpoint Usage**: Which endpoints are used most frequently
- **Response Times**: Average and peak response times
- **Error Rates**: Percentage of failed requests

<Info>
  API usage is tracked separately from web dashboard usage but counts toward the same quota limits.
</Info>

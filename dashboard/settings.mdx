## Setting Up AI Providers

One of the most important settings is configuring your AI providers. Follow these steps to set up your AI model access:

<Steps>
  <Step title="Navigate to AI Model Providers">
    Go to Settings → General → AI Model Provider
  </Step>
  <Step title="Obtain API Keys">
    Get API keys from your preferred providers:
    <ul>
      <li><b>OpenAI:</b> [https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)</li>
      <li><b>Mistral AI:</b> [https://console.mistral.ai/api-keys/](https://console.mistral.ai/api-keys/)</li>
      <li><b>Google AI (Gemini):</b> [https://makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)</li>
    </ul>
  </Step>
  <Step title="Enter API Keys">
    Add your API keys to the corresponding fields in the settings panel
  </Step>
  <Step title="Select Default Models">
    Choose your preferred default models for different content types
  </Step>
  <Step title="Test Connection">
    Use the "Test Connection" button to verify your API keys are working correctly
  </Step>
  <Step title="Save Configuration">
    Click "Save Changes" to apply your AI provider settings
  </Step>
</Steps>

<Warning>
  **Security Note:** Your API keys are sensitive credentials. ReplyIQ encrypts these keys in storage and never shares them with third parties. We recommend using API keys with appropriate usage limits and rotating them periodically.
</Warning>

## Managing Your Subscription

<AccordionGroup>
  <Accordion title="Viewing Current Plan">
    Your current subscription details are displayed in the Subscription Management section, including:
    - Plan name and features
    - Monthly/annual cost
    - Usage limits
    - Renewal date
    - Payment method
  </Accordion>
  
  <Accordion title="Upgrading Your Plan">
    To upgrade your subscription:
    1. Navigate to Settings → Subscription Management → Plan Options
    2. Compare available plans
    3. Select your desired plan
    4. Review the changes in features and pricing
    5. Confirm your upgrade
    6. Receive confirmation of your new subscription
  </Accordion>
  
  <Accordion title="Updating Billing Information">
    To update your payment details:
    1. Go to Settings → Subscription Management → Billing Information
    2. Select "Edit Payment Method"
    3. Enter your new payment information
    4. Save your changes
    5. Verify the update was successful
  </Accordion>
</AccordionGroup>

## Security Best Practices

<AccordionGroup>
  <Accordion title="API Key Management">
    - Use separate API keys for different environments
    - Set usage limits on your API keys
    - Rotate keys periodically (every 90 days recommended)
    - Never share keys in public repositories or forums
  </Accordion>
  
  <Accordion title="Account Security">
    - Use a strong, unique password
    - Enable two-factor authentication
    - Review login history regularly
    - Log out from shared devices
  </Accordion>
  
  <Accordion title="Data Protection">
    - Review and adjust data retention settings
    - Export and delete sensitive data when needed
    - Understand how your content is stored and processed
    - Review third-party access permissions regularly
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="API Connection Issues">
    If you're experiencing problems with AI providers:
    - Verify your API key is entered correctly
    - Check your API key has sufficient credits/quota
    - Ensure your account is in good standing with the provider
    - Test with a simple request to isolate the issue
  </Accordion>
  
  <Accordion title="Billing Problems">
    For subscription or payment issues:
    - Verify your payment method is up to date
    - Check for any declined transactions
    - Review your invoice for discrepancies
    - Contact support for billing assistance
  </Accordion>
  
  <Accordion title="Settings Not Saving">
    If settings aren't being saved:
    - Clear your browser cache
    - Try a different browser
    - Disable browser extensions
    - Check for network connectivity issues
  </Accordion>
</AccordionGroup>
